# ToDos

A simple application to help you manage tasks, goals, or activities you need to complete.

## Technology
- Vite
- React.js
- React Router
- Redux
- Redux Toolkit
- TypeScript
- Material UI

## Features
- Add & delete ToDos
- Mark ToDos as completed
- Home navigation route
- ToDos list route
- Create ToDo form route
- Responsive design
